2 вида обращения к свойству.
value.property - точное название свойства
value[property] - может находиться выражение

Элменты массива хранятся как совйства массива.

Объект
var day1 = {
...: ...,
...: ...
};

delete day1.property - удаление свойства
Object.keys(объект) - вернет все имена свойств объекта

property IN object - возвращает true, если объект содержит это свойство

Примитивы в js являются неизменяемыми.
Метод slice для извлечения подмассива из массива.

function SetA(...numbers), ...numbers equal params int[] numbers
При вызове массив можно наоборот преобразовать в элементы в виде отдельных аргументов.
SetA(...numbers);
