Глава 15. События.
У событий срабатывает всплытие, то есть при срабатывании события на дочернем элементе затем срабатывает аналогичное событие для всех предков. Чтобы это избежать можно вызвать stopPropagation.
Либо можно при помощи event.target проверять какой тип элемента вызвал событие.
Отмена события по умолчанию после кастомного обработчика событий - preventDefault
Для считывания нажатой клавиши при вводе текста лучше использовать событие input, а не keydown, так как keydown может неправильно распознавать символы из разных источников ввода.
При завершении загрузки страницы срабатывает событие load. Используется для инициализации.

Таймеры.
setTimeout(func, time) - запустит func через time мс.
cleatTimeout(timeout) - отмена запланированной функции.
setInterval() - повторять func каждые time мс.
