Глава 3. Функции.

Return, который не возвращает значения, присваивает переменной значение undefined.

Определение функции
let showName = function(name){
console.log(name);
};

Объявление функции
function showName(name){
console.log(name);
}

Объявленная функция доступна в самом начале программы, а определенная функция толко в коде после определения функции.

При передачи большего количества аргментов функции, чем параметров в функции, избыточные аргументы игнорируются. Если передать меньше - то непереданные будут равны undefined.

Замыкание.
Для каждого нового вызова основной функции создается новая локальная переменная, но если внутри функции возвращается другая функция, которая использует локальную переменную внешней функции, то эта переменная замыкается и остается в памяти до тех пор, пока на внутреннюю переменную существут ссылка.
